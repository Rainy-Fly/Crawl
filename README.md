# 整体架构 
这是一个基于python,html,css,js的爬虫系统
每个用户各自的有账号密码，后端有数据库存储用户信息，存储内容包含：爬虫配置文件，时间配置文件，保存的爬虫结果
# sign.html页面，用户可以注册账号，登录账号,登录后来到index.html
# index.html页面，有以下功能区：
## 上传爬虫配置文件，时间配置文件
上传方式有：页面上有个框可以编写json,也可以点击上传按钮上传json文件
## 查看已有爬虫和时间配置文件
在一个scroll中有用户已经上传的爬虫配置文件和时间配置文件，点击后能显示详细内容并编辑修改
## 查看示例配置文件
在页面上有个按钮，点击后能显示示例配置文件（crawl_demo_config.json,time_demo_config.json）
点击后能下载示例配置文件到本地
## 爬虫选项
选择一个爬虫配置文件，可选时间配置文件，不选的话立即开始爬虫，否则按照时间配置定时启动，点击启动按钮后进入爬虫池
## 爬虫状态显示区域
正在运行的爬虫状态，显示未开始/进行中/已完成/已暂停
已完成的爬虫后面有两个按钮，1.可视化，到show.html,2.保存，保存到数据库
# show.html页面，用户可以查看爬虫结果，这是可视化
最左侧的scroll中有数据库中已经保存的爬虫结果，以及已完成但没有保存的爬虫结果，点击后可以更改可视化展示的内容
中间是主要的可视化区域，展示当前选中的爬虫结果
# 后端分层架构
数据库增删查改与用户登录
基于flask框架的路由，每个路由对应一个功能，如上传文件，查看配置文件，查看状态，可视化展示，保存结果等
